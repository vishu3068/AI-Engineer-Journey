# Functions
A function is a reusable block of code.
def is used to create a function.
A function is executed by calling it.
Parameters allow values to be passed into a function.
return sends a value back to the caller.

# Tuples
A tuple stores multiple values.
Tuples are immutable (cannot be changed after creation).
Tuples are created using parentheses ().
Useful methods: count(), index(), len().

# Sets
A set stores unique values.
Duplicate values are automatically removed.
Sets are unordered.
Common methods: add(), remove(), len().
Membership can be checked using the in operator.

# Day 5 Notes

## Strings

- String stores text data.
- Written inside quotes.
- Index starts from 0.
- len() returns length of string.
- upper() converts string to uppercase.
- lower() converts string to lowercase.

Example:

name = "Vishwanth"

print(name[0])

Output:
V

---

## Lists

- List stores multiple values.
- Written inside square brackets [].
- Index starts from 0.
- append() adds new item.
- len() returns total items.

Example:

fruits = ["Apple", "Mango"]

fruits.append("Banana")

Output:

['Apple', 'Mango', 'Banana']

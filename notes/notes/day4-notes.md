FOR LOOP

Used when number of iterations is known.

Syntax:

for variable in range(number):
    statement

Example:

for i in range(5):
    print(i)

----------------------------------

WHILE LOOP

Used when condition controls repetition.

Syntax:

while condition:
    statement

Example:

count = 1

while count <= 5:
    print(count)
    count += 1

# Day 11 — OOP Inheritance

## What is Inheritance?
inhertence is a feature in oops which allow to inherit the methods and properties of one class to another
## Parent Class
main class its methods or properties are inherited by other class
## Child Class
sub class it inherits the methoda and properties of other class
## Inherited Method
inherited method can be defined as the method of parent class used by child class
## Child's Own Method
child class can create its own method that method is called as the child own method
## Method Overriding
when parent and child class hase same method name but diff functions then child method overides the parent method 
## What is super()?
super() is a feature which allows a child class to call methods from its parent class
## Why use super().__init__()?
to in herit the attributes from parent class and specifically calls the parent constructor so the child can initialize inherited attributes.
